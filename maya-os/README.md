# Maya OS

A personal productivity OS — dark gamer/cyberpunk aesthetic, gamified task management, built for daily active use.

## What it is
- **Day view** — task queue with frogs (must-do items), spotlight zone for focused/timed work, daily habit tracker
- **Week view** — 7-day grid, drag tasks to reschedule, score overview per day
- **Stats view** — XP level, streaks, habit consistency, point targets
- **Gamification** — close your day to earn XP, gain levels, streak multipliers, momentum tracking
- **Natural language task input** — type `frog Fix the bug 2 45m` to create a frog task worth 2pts with 45m estimate

## Tech
Vite + React 18, plain CSS modules, localStorage only (no backend).

## Run it
```bash
npm install
npm run dev    # http://localhost:5173
```
